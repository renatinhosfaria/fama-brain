# Simulações de Referência — ICP Fama

Cenários padrão de simulação para o ICP da Fama (Uberlândia/MG, econômico/MCMV). Use estes como base para primeira conversa com o cliente — valores aproximados, confirmar no simulador CEF antes de oferta formal.

**Última verificação: 2026-04-21.** Taxas e parâmetros MCMV das novas regras em vigor a partir de 22/04/2026.

## Premissas comuns

- **Local:** Uberlândia/MG (faixa populacional 300–750 mil hab.).
- **Modalidade:** MCMV via Caixa, SAC + TR.
- **Prazo:** 360 meses (30 anos) — prazo máximo em MCMV é 420 meses (35), mas 360 é padrão comercial.
- **Entrada mínima:** 10% do valor do imóvel (exceto quando indicado).
- **FGTS:** cliente com saldo usado na entrada.

## Cenário A — Faixa 1

**Perfil:** casal jovem, renda conjunta R$ 2.800/mês (ou solo até R$ 3.200), primeiro imóvel, 5 anos de FGTS com saldo de R$ 10.000.

| Item | Valor |
|---|---|
| Valor do imóvel | R$ 200.000 |
| Entrada | R$ 20.000 (10%) |
| FGTS na entrada | R$ 10.000 |
| Recursos próprios | R$ 10.000 |
| Valor financiado | R$ 180.000 |
| Subsídio estimado | até R$ 55.000 |
| Taxa | 4,50% a.a. + TR |
| Prazo | 360 meses |
| **1ª parcela aproximada** | **R$ 1.050 – R$ 1.150** |
| Comprometimento de renda | ~37% (limite 30% geral, MCMV aceita até 30-35%) |

**Observação:** subsídio direto reduz o valor financiado — parcela efetiva pode ser menor. Confirmar no simulador.

## Regras rápidas de viabilidade

- **Comprometimento de renda:** máximo 30% em geral; MCMV aceita até ~35% em alguns casos. Se passar, reprovação quase certa.
- **Score/SPC-Serasa:** cliente com score baixo (<400) ou restritivo provavelmente será reprovado. Antecipar a regularização.
- **Vínculo de trabalho:** CLT é mais simples; PJ/autônomo precisa de 2 anos de IR + extratos bancários.
- **Idade máxima do devedor no fim do contrato:** 80 anos e 6 meses (limite CEF). Compor com um mais novo se preciso.

## Como usar

1. Levantar: renda, FGTS, score estimado, primeiro imóvel sim/não, valor alvo.
2. Bater com o cenário mais próximo acima.
3. Rodar simulação oficial no site da Caixa ou app correspondente para confirmar.
4. Comparar com SBPE em ao menos 1 banco privado se o cliente estiver na Faixa 4 ou fora do MCMV.

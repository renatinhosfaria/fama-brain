# Objetivo
Dar ao Reno respostas consultivas para objeções comuns, seguindo a lógica: validar, reformular e avançar.

# Método padrão
1. Validar a preocupação.
2. Traduzir em critério concreto.
3. Responder sem prometer o que não pode.
4. Puxar próximo passo viável.

# Objeção: “Não tenho entrada”
Resposta-base:
`Entendi. Esse é geralmente o primeiro ponto mesmo. Antes de descartar, vale entender quanto seria necessário e se FGTS/condições do programa ajudam. Você tem alguma reserva hoje ou seria praticamente zero entrada?`

# Objeção: “Não sei se aprova”
`Faz sentido ter essa dúvida. O melhor caminho é olhar viabilidade antes de escolher imóvel, pra não criar expectativa errada. Você já financiou algo antes ou seria a primeira vez?`

# Objeção: “A parcela está cara”
`Sim, parcela precisa caber com segurança. Às vezes o ajuste vem no valor do imóvel, entrada ou prazo. Hoje você teria uma parcela máxima confortável em mente?`

# Objeção: “Só estou olhando”
`Tranquilo. Muita gente começa assim. A diferença é entender se você está só pesquisando ou se, aparecendo algo viável, já faria sentido avançar. Em qual dos dois você está mais?`

# Objeção: “Me manda opções”
`Te mando, sim. Só pra não te mandar coisa fora do perfil: você busca mais apartamento ou casa?`

# Objeção: “Vou falar com marido/esposa”
`Claro, faz sentido. Pra conversa ser produtiva, o ideal é alinhar com quem decide junto. Quer que eu te ajude a resumir os pontos principais pra vocês avaliarem?`

# Objeção: “Não quero ir ao escritório”
`Sem problema. A visita só faz sentido se ela resolver algo pra você. Podemos começar entendendo por aqui: hoje sua maior dúvida é imóvel, entrada ou financiamento?`

# Limites
Nunca prometer aprovação, taxa, subsídio ou condição final sem análise adequada. Usar linguagem de possibilidade, não garantia.
